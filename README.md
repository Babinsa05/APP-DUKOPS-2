# DUKOPS PREMIUM - Koramil 1609-05/Sukasada

Aplikasi DUKOPS untuk memudahkan Babinsa dalam membuat laporan harian.

## Fitur

- ✅ Pilih Babinsa (desa/kelurahan)
- ✅ Upload multiple foto
- ✅ Watermark otomatis (tanpa kata Desa/Kelurahan)
- ✅ Kompresi gambar (resize 1280px, quality 80%)
- ✅ Pilih tanggal dan jam (detik otomatis)
- ✅ Narasi otomatis dengan format lengkap
- ✅ Kirim ke Telegram
- ✅ Simpan ke Google Drive (auto-create folder Tahun/Bulan/Desa)
- ✅ PWA - bisa diinstall di Android
- ✅ Halaman offline

## Cara Install di Android

1. Buka `https://babinsa05.github.io/APP-DUKOPS-2/` di Chrome
2. Klik menu (3 titik) → "Install App" atau "Add to Home screen"
3. Aplikasi akan terinstall seperti APK biasa

## Teknologi

- HTML5, CSS3, JavaScript
- PWA (Service Worker, Manifest)
- JSZip untuk kompresi file
- Google Apps Script backend
- Telegram Bot API
- GitHub Pages hosting
